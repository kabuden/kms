# 서술형 문제모음

## 1. 일반 및 기타
```dataview

table without id
	file.link as "문제번호",
	관련노트,
	중요,
	반복도,
	풀이
	
from "(21) 문제모음"

where
	contains(tags,"✏️")
where
	contains(Category1,[[1. 일반 및 기타]])	
where	
	중요 < 11
sort 반복도 desc

```
